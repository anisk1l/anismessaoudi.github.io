# Static Site Generators

An SSG builds static HTML pages from plain text (Markdown) files.

## How it works
1. You write content in Markdown
2. The SSG applies an HTML template
3. Static HTML pages are generated and served

## Popular SSGs
- **Jekyll** — Ruby, powers GitHub Pages
- **Hugo** — Go, extremely fast
- **Next.js** — JavaScript with React templates
- **Eleventy** — supports 10 template languages
