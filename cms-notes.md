# Content Management Systems

A CMS is software used to manage the creation and modification of digital content.

## Key Milestones
- **1990–1995** — Static HTML only, no CMS
- **1995–2000** — Server-side scripting (PHP, ASP, JSP)
- **2000s** — Open-source CMS: WordPress, Drupal, Joomla
- **2010s** — Headless CMS with API-only backends

## Types of CMS
- **Web CMS** — WordPress, Drupal
- **LMS** — Moodle (learning management)
- **COMS** — ConfTool (conference management)
